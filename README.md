**[SecureAttend](https://github.com/hasansyed18)** — offline-first attendance verification that eliminates proxy attendance by cross-checking BLE proximity, RSSI filtering, and geofencing before marking anyone present.
`BLE` `RSSI` `Geofencing` `Firebase` `Android` · 1st Prize, IETE South Zone 2026

**[GridMap AI](https://github.com/hasansyed18)** *(in progress)* — a grid-based visual map builder that converts floor layouts into graph-based navigation models with accessibility-aware routing, removing the need for CAD files.
`Flutter` `Firebase` `Graph Algorithms` `Indoor Positioning` · architecture & Phase 1 in progress

`NodeMCU` `MQ6` `Firebase` `JavaScript` · KSCST State-Level Selection
