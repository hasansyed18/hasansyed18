### Featured Projects

**[SecureAttend](https://github.com/hasansyed18) — Multi-layer presence verification system**
Offline-first attendance verification built to eliminate proxy attendance, using BLE proximity, RSSI signal filtering, and geofencing. Devices are bound to a single verified user, with automatic Firebase sync once connectivity returns.
`BLE` `RSSI Filtering` `Geofencing` `Firebase` `Android`
→ 3 of 7 phases delivered · 1st Prize, IETE South Zone Seminar & ISF Congress 2026

**[GridMap AI](https://github.com/hasansyed18) — Self-service indoor navigation platform** *(in progress)*
A visual, grid-based map builder that lets institutions create interactive indoor maps without CAD files. Grid layouts convert automatically into graph-based navigation models with accessibility-aware routing. Phase 1 is scoped around QR-checkpoint indoor positioning and multilingual voice-guided navigation.
`Flutter` `Firebase` `Graph Algorithms` `Indoor Positioning`
→ Architecture and Phase 1 scope in progress

**[Smart LPG Gas Leak Detection System](https://github.com/hasansyed18) — IoT safety system**
Real-time gas leak detection built on an ESP8266 NodeMCU and MQ6 sensor, with push notifications and remote motor control for immediate hazard response.
`NodeMCU / ESP8266` `MQ6 Sensor` `Firebase` `HTML/CSS/JS`
→ KSCST State-Level Selection · multiple project exhibition awards
